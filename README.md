# AI-Powered VPN Support Chat Terminal (Microsoft Foundry Edition)

An enterprise-ready, high-concurrency support dashboard engineered using Next.js and the Microsoft Azure AI Projects SDK framework.

## Tech Stack
- **Framework:** Next.js (App Router)
- **AI Core Runtime:** Microsoft Azure AI Projects SDK (`@azure/ai-projects`)
- **Package Manager:** pnpm
- **Identity & Auth:** `@azure/identity` (DefaultAzureCredential)
- **Styling:** Tailwind CSS + Lucide Icons
